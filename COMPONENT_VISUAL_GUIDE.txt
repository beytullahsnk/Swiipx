        │

        │

        │
  0.5s ─┼─ Benefit 3 fades in
        │
  0.6s ─┴─ Benefit 4 fades in


User Interactions:
  Thumbnail click      → Instant selection (ring appears)
  Pack selection       → 300ms border/background transition
  Add to cart click    → Scale animation + toast notification
  Collapsible toggle   → 300ms height animation

╔════════════════════════════════════════════════════════════════════════════╗
║                            STATE MANAGEMENT                                ║
╚════════════════════════════════════════════════════════════════════════════╝

useState hooks:
  
  selectedPack: number
  ├─ Default: 2 (pack 2)
  ├─ Range: 1-3
  └─ Updates: onClick of pack buttons
  
  selectedImage: number
  ├─ Default: 0 (first thumbnail)
  ├─ Range: 0-3
  └─ Updates: onClick of thumbnails
  
  expandedSection: string | null
  ├─ Default: null (all collapsed)
  ├─ Values: 'how-it-works' | 'delivery' | 'guarantee' | null
  └─ Updates: onClick of section headers

║                            RESPONSIVE BREAKPOINTS                          ║
╚════════════════════════════════════════════════════════════════════════════╝

Mobile:         < 640px   (sm)  │  Single column, stacked
Tablet:    640px - 1024px (md)  │  Still stacked, larger text
Desktop:      ≥ 1024px    (lg)  │  2-column layout activated

Tailwind classes used:
  sm:text-4xl     → 640px+
  lg:grid-cols-2  → 1024px+ (2-column layout)
  lg:gap-16       → 1024px+ (larger gap)
